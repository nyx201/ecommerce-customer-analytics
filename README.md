# E-Commerce Customer Segmentation & Revenue Analysis

**Tools:** MySQL · Python · Power BI  
**Dataset:** UCI Online Retail II (500K+ transactions)  
**Domain:** Retail Analytics | Customer Intelligence

## Project Overview
Analyzed 500K+ transactions from a UK-based online retailer to segment 
customers using RFM methodology and K-Means clustering, and built an 
interactive Power BI dashboard for business decision-making.

## Key Findings
- Top 15% of customers (Champions) drive majority of total revenue
- UK accounts for 80%+ of transactions
- K-Means identified 4 distinct customer behavioral clusters
